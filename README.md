# chat-robot
chat-robot一个简单的聊天机器人的入门，目前：
+ 1、使用koa+vue，简单的聊天实现

# 栗子1：chat-robot-koa-vue
1、启动
在chat-robot-koa根目录下，执行`npm i`，然后执行`npm run start`;

在chat-robot-vue根目录下，执行`npm i`，然后执行`npm run dev`;

访问 http://localhost:8080
